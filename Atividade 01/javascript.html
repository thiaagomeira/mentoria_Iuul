JavaScript

Site Oficial: MDN Web Docs

Definição: JavaScript é uma linguagem de programação interpretada, usada para criar páginas web dinâmicas e interativas. É uma das três principais tecnologias da web, ao lado de HTML e CSS.

Prós

  • Ampla compatibilidade com navegadores.
  • Alta velocidade de execução.
  • Grande comunidade e suporte.

Limitações

  • Problemas de segurança.
  • Executa no lado do cliente, o que limita o controle sobre o ambiente de execução.
  • Difícil depuração em grandes projetos.
